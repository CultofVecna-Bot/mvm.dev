# 什么是 MVM

Mixin 的愿景是保护用户的消息和数字资产的隐私和安全，由于 Mixin 采用隐私程度相同但更灵活 TIP 方案来存储私钥，使得其它链（比如 ethereum）的开发者不能方便的使用已有的技术方案。MVM 就是为了方便合约开发者基于 Mixin 开发。

MVM 可以让其它网络上的开发者（几乎）不需要做任何修改，直接迁移应用到 Mixin 上使用。
